<Bigger><Bold>Mail Folders</Bold></Bigger>


Mail folders will show up as icons along the top of the main window. The messages in the active folder will show up in a list below that.


The <Bold>Folder</Bold> menu contains commands that allow you to create and open folders and to switch between (activate) them. Most of the commands expect you to first select one or more folder icons. An icon can be selected with a single click. Several icons can be selected by dragging the pointer or by holding down the Shift or Control keys while clicking. You can also activate a folder by double-clicking on its icon, or by holding down the right mouse button and using the popup menu.


<Bigger><Bold>Reading Messages</Bold></Bigger>


The <Bold>Message</Bold> menu contains commands that allow you to compose, read, save, print and reply to messages. Like the <Bold>Folder</Bold> menu, most of the commands expect you to first select one or more messages. For example, to read a message you can:


<Indent>Double-click on the summary line for that message. <Italic>-or-</Italic>

Single-click <Indent>on the summary line for that message to select it, then use the <Bold>Read</Bold> command in the <Bold>Message</Bold> menu. <Italic>-or-</Italic></Indent>

Hold <Indent>down the right mouse button over the message summary line to display a popup menu for that message. Use the <Bold>Read</Bold> command in the popup menu.</Indent>
</Indent>


<Bigger><Bold>Composing Messages</Bold></Bigger>


When you are <Bold>composing</Bold> a new message, you are typing into a <Italic>Motif</Italic> text widget. The editing commands used by this widget can be found in the <Underline><Bold><Italic>Ishmail</Italic> User's Guide</Bold></Underline>. If you wish to use your own editor, check out the <Bold>Editor</Bold> function under the <Bold>Message</Bold> menu in the composition window.


<Bigger><Bold>Customization</Bold></Bigger>


<Italic>Ishmail</Italic> is highly customizable to your own tastes and habits. The <Bold>Options</Bold> menu in the main window contains commands that let you specify custom beeps and icons, pick the fonts for your display, add buttons to your interface, and control the appearance of your message list among other things.

<Bigger><Bold>Pictures and Sounds</Bold></Bigger>


This section shows how <Italic>graphics</Italic> and <Italic>sounds</Italic> can be included in a message. The display of these components is through external commands that you choose. Ishmail comes with a <Bold>mailcap</Bold> file containing lines of the form:


<Indent><Fixed>image/gif; xv -perfect %s</Fixed></Indent>


This line tells <Italic>Ishmail</Italic> to use the <Bold>xv</Bold> command to display GIF components. If you do not have the <Bold>xv</Bold> command, or you wish to use a different command, create a file in your <Bold>$HOME</Bold> directory called <Bold>.mailcap</Bold>. Add a line like the above one for <Italic>image/gif</Italic> and change the command to one of your choosing. Add the <Bold>%s</Bold> token where the name of the file would normally go.
